HoverTabOfLink

链接预览悬浮窗工具

一款现代化的浏览器链接预览工具，支持 Chrome/Edge/Firefox，采用 Manifest V3 标准。 提供流畅、高效、可高度自定义的悬浮窗口预览体验，帮助用户在不离开当前页面的情况下快速获取链接信息。 支持拖拽文本搜索、多种触发方式和多语言界面，全面提升您的浏览效率。

🚀 立即安装

🚀 核心特性

🔍

链接预览

通过预设的触发方式（如悬停、拖拽链接等），在当前页面上打开悬浮窗口，快速预览链接内容。悬浮窗支持固定、刷新和在新标签页中打开。

🖱️

多种触发方式

支持拖动链接、鼠标悬停、长按、Alt+鼠标左键点击、Alt+鼠标悬停等多种触发方式，满足不同用户的使用习惯，并可自定义快捷键。

🎛️

高度自定义

可调整悬浮窗大小、位置、背景遮罩透明度等，提供个性化的预览体验。支持系统主题、浅色主题和深色主题切换。

📐

灵活调整

悬浮窗支持拖拽移动和调整大小，可固定在屏幕任意位置，方便对比查看内容。新增自动固定功能，适合多任务处理场景。

🌐

多浏览器支持

完美支持 Chrome、Edge 等主流基于 Chromium 的浏览器，一次开发，处处可用。智能适配不同操作系统，提供一致体验。

🔤

拖拽文本搜索

支持拖拽页面选中文本进行搜索，可选择必应、谷歌、百度、DuckDuckGo等搜索引擎。可设置禁用网站列表，最多支持50个域名。

🌍

多语言支持

支持简体中文、繁体中文、英语、日语、韩语、法语、德语、俄语、意大利语、西班牙语、葡萄牙语、阿拉伯语等12种语言，覆盖全球主要地区用户。

📊

资源管理

可设置同时打开的悬浮窗数量（1-6个），当达到上限时自动关闭最早打开的悬浮窗，帮助用户有效管理系统资源，避免过多悬浮窗影响浏览体验。

⚙️

现代化架构

基于 Manifest V3 标准，使用 TypeScript 和 React 开发，代码质量有保障。采用 WXT 框架、Tailwind CSS 和 shadcn/ui 组件库，提供流畅的用户体验。

⚙️ 设置选项

🔗

链接预览设置

自定义触发方式、悬停延迟、长按延迟、弹窗大小和位置、背景遮罩透明度、悬浮窗数量限制和自动固定功能，打造个性化的预览体验。

✋

拖拽文字设置

启用或禁用拖拽文本功能，选择默认搜索引擎（必应、谷歌、百度、DuckDuckGo），设置禁用网站列表，支持通配符和域名格式验证。

🎨

主题与语言

选择系统主题、浅色主题或深色主题，切换界面语言（支持12种语言），提供舒适的视觉体验和本地化界面。

🛠️ 安装使用

Chrome/Edge 安装步骤

1

下载扩展

从 Chrome 应用商店搜索并安装 HoverTabOfLink 扩展。

2

开始使用

安装完成后，在任意网页上尝试拖动链接或悬停，即可看到悬浮预览窗口。

3

个性化设置

通过扩展侧边栏访问设置面板，根据个人喜好调整各种选项。
